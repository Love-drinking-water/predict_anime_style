运行环境：Python 3.7.6, CUDA 10.0, Tensorflow-gpu 2.0.0, Keras 2.3.1, Ubuntu 18.04.

data/
|-- 动画信息.xlsx # 数据集，数据从 https://bangumi.tv/ 爬取，变量解释如下：
|------ 动画ID，可唯一标识一部动画。
|------ 动画名，动画的标题。
|------ 动画封面，对应动画封面的文件名，文件被存放在动画封面的文件夹中。
|------ 动画简介，该动画的简介，已经使用百度翻译API处理为简体中文。
|------ 第一风格**（5~14列），该动画的最显著风格，1代表是，0代表否，本模型没有使用这些变量。
|------ **（15~24列），该动画是否拥有这个风格，本模型使用的因变量是这十列中的某一列。
|------ 有效前作，该动画的前作动画ID，如果无前作动画则取空值。（例：进击的巨人第二季的前作动画为进击的巨人第一季）
|------ 主线故事，该动画的主线故事动画ID，如果无主线故事动画则取空值。（例：名侦探柯南-绀青之拳的主线故事动画为名侦探柯南）
|------ 全为男性，该动画的主角是否全为男性。（同时需要满足主角数量>=2，下同）
|------ 全为女性，该动画的主角是否全为女性。
|------ 男多于女，该动画的主角是否男多于女。
|------ 女多于男，该动画的主角是否女多于男。
|------ 男女相等，该动画的主角是否男女相等。
|-- 动画封面        # 文件夹，用于存储动画封面。
|-- 用于预测的动画信息.xlsx    # 数据集，2022年1月的5部动画，不在上面的数据集中，没有因变量，用于模型预测未来的尝试。模型预测结果已经输出到该表格。
|-- 用于预测的动画封面 # 文件夹，用于存储动画封面。
|-- simhei.ttf # 字体，微软雅黑。
|-- cn_stopwords.txt # 停止词，用于绘画词云图。

code/
|-- 动画风格条形图.ipynb # 数据可视化代码，统计动画风格。
|-- 标题与简介的词云图.ipynb # 数据可视化代码，绘画标题和简介分词后的词云图。
|-- 模型代码.ipynb # 模型的核心代码，用于训练、验证、测试模型。
|-- 模型效果重现及预测.ipynb # 根据已经训练好的模型重现演示文稿中的模型结果，并且对未来做出预测。

result/
|-- 训练好的模型 # 文件夹，存放根据 code/模型代码.ipynb 训练出的模型，用于模型效果重现。
|-- 数据可视化图片 # 文件夹，存放各类可视化结果，包括动画风格条形图、分词后简介标题的长度直方图、词云图，以及各个风格各个模型的ROC曲线。
